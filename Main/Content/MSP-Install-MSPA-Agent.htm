Install the MSP Anywhere remote agent

Download and install the MSP Anywhere remote agent from within to launch remote sessions for your customers devices. After the remote agent is installed, you can access any asset to remotely diagnose your customers' issues. Only one of these tasks is necessary to install the MSP Anywhere remote agent.

Download the MSP Anywhere Remote Agent from the Customer Dashboard

  2. From the Customer Dashboard, download the Remote Access Agent Installer.
  3. Copy the Remote Install Link.
  4. Send the Installation Link to your customer through email, or select a contact in the Email Contacts list.
  5. Click Send to forward the installation link to your customer.

Download the MSP Anywhere Remote Agent from inside an Asset

  1. Click Knowledge > Assets > Select an asset in the list.
  2. Click MSP Anywhere to view options.
  3. Copy the Remote Install Link.
  4. Send the Agent Installer URL to your customer through email, or select a contact in the Email Contacts list.
  5. Click Send to forward the installation link to your customer.
