Import and sync Assets from your MSP Anywhere account

After you import assets to your account, you can quickly start a remote session for any device in your customer's network to take a closer look at reported issues.

  1. Select a customer > Assets.
  2. Click Add A New Asset > Import from MSP Anywhere.
  3. Select the assets to import from the Mapped Asset Name that correspond to each MSP Anywhere asset.
  4. Select the check box for each asset you want to import.
  5. Click Import Assets and the status is displayed in the Import Status column.
  6. Select the check box next to each asset you want to ignore.

    Ignore any assets you do not want to import, these will remain in MSP Anywhere but will not be imported into .

  7. Click Ignore Assets and the status is displayed in the Import Status column.
