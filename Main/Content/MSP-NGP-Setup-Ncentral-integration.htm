Configure the N-central integration in

You can import and map N-central assets and devices to service items. N-central updates these devices in any time a change is detected in your network so you can bill for the most current count of devices each month.

  1. Navigate to Settings > Integrations > N-central > Configure.
  2. Click Add MSP Manager Credentials.
  3. Enter the necessary credentials and click Save.

    4. These should match your PSA Integration Credentials in N-central, and are only used for the integration with N-central.

  4. Click Add N-central Credentials.
  5. Enter the required information.

    6. These credentials should match your N-central Admin login information, and are required for to pull information from N-central.

  6. Click Save.
