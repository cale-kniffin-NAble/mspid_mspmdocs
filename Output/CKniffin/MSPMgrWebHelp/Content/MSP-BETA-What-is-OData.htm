MSP Manager

Use OData to export your MSP Manager data

Odata is an export endpoint that can feed real-time data into Excel or Power BI. Use the OData feature in Microsoft Excel or Power BI to export data from MSP Manager.

Make sure you are using Excel 2016 or newer. If you are using an older version of Excel, you will need to download Microsoft Power Query to use the OData option.

  1. From Excel, navigate to the Data tab and click New Query > From Other Sources > From OData feed. From Power BI, click the Home tab and click Get Data > OData feed.
  2. If you are using an older version of Excel and do not have the New Query option listed in the Data tab, download and install Microsoft Power Query and restart Excel. When Excel re-opens, click the Power Query tab and click From Other Sources > From OData Feed.
  3. In the OData Feed box, enter the URL https://api.mspmanager.com/odata and click OK.
  4. Select Basic from the options on the left, enter your MSP Manager login credentials and click Connect.
  5. In the OData Navigator window, select which data type you would like to export from MSP Manager and click Load.
  6. To import additional data, open a new sheet in Excel or a new page in Power BI. Select Recent Sources from the Data tab in Excel or the Home tab in Power BI, and choose the api.mspmanager.com/odata connection that was previously established. From the Navigator window, select the data type you would like to import and click Load.
  7. Repeat this process as needed to export additional data.
