Organize your tickets with ticket queues

Create ticket queues to prioritize work for teams of technicians. These queues can help group tickets based on the type of issue, or the technicians most qualified to handle the ticket. Add tickets to those queues for the assigned users to monitor and work from.

Create ticket queues

  1. Click Account Settings > Queues > Add New Queue.
  2. Enter a Name and Description, and mark the queue as Active.
  3. Click Add Assigned Users.
  4. Save the queue.

Create as many Queues as needed, and start adding tickets to them.

Add tickets to a queue

  1. Click Helpdesk, and select a ticket.
  2. Click anywhere in the Ticket Information section to expand and edit the ticket.


  3. Select a Queue.


  4. Save the ticket.

Use the Queue filter in the Helpdesk to view all tickets in a specific queue.
