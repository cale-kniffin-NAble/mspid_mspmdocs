Import existing passwords into custom knowledge templates

After creating a custom knowledge item with a Credential Field Type, you can import existing passwords into the new knowledge template when applying it to customers.

The linked password will be removed from the Knowledge > Legacy Knowledge > Passwords section and moved to the new knowledge article.

Apply the Credential Knowledge Type to customers and link it to existing passwords

  1. Follow the steps to create a custom knowledge item with a Credential Field Type.
  2. Navigate to Knowledge > All Knowledge, click and select the new Credential knowledge type.
  3. Next to the Customer Field, above the Credential section of the knowledge item, select Link existing to link an existing password to the new knowledge item.
  4. Use the Search field to find an existing password or select a password from the list, and click to link the password.

The Credential section of the knowledge item is auto populated with the imported password information, and the linked password is removed from the original Passwords section of the Knowledge area.
