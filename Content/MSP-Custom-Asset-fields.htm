Create Custom Asset Class fields

Customize Asset Class Fields to capture and better understand your customers' environment and business practices when it comes to how they track and store servers, workstations, devices and other assets.

Apply Asset Permissions to users

Before you get started, make sure users have the appropriate permissions to access and create Custom Asset Class fields.

  1. Navigate to Settings > Users & Permissions > Roles.
  2. Select to edit the roles that require permissions for custom asset fields.
  3. Select the Asset Management section to expand it's options.
  4. Select the permissions to apply to the role or select All Asset Management to apply all permissions. Select Save Role when finished.

Create Custom Asset Class fields

  1. Navigate to Settings > Assets > Asset Classes > Add Asset Class.
  2. In the General Information section, enter a name and description for the new Asset Class.
  3. Select Add new to add a custom field to the asset.
  4. Select a Field Type from the list. Each Field Type contains additional Field Settings:
    • Checkbox - add a Field label that is displayed next to the checkbox.
    • Date - add a Field label and a Placeholder, like the suggested date format (i.e. mm/dd/yyyy).
    • Dropdown - in addition to a Placeholder and Field Label, add options to be included in the dropdown menu. Press enter, or select Add new option to add additional dropdown menu items.

      • Select Is required to make this a mandatory item in the asset.

    • Numeric - add a Field label and a Placeholder that is displayed in the text box prior to the user entering information.
    • Rich Text - enter a Field label. This field allows for things like bold font, bullet pointed or numbered lists, hyper links and inserted images.

      • Use drag and drop functionality to imbed images into rich text fields. Do not copy and paste as this will display a broken' image.

    • Section - enter a Field label and select a Section Layout. This allows for the addition of sub fields within each area of the Section Layout. Click Add new in each section to add additional fields.
    • Text - the requirements for this field are the same as the Numeric field with the additional option to select whether the field is required or not.
  5. Continue to add custom fields as needed, and select Save Field to confirm the addition of each new field.
  6. Select Save Asset Class when finished.

Add Custom Asset fields to customers' Assets

After creating Custom Asset Class fields, you can apply them to a customer's asset.

  1. Navigate to Assets and select next to the asset you want to edit.
  2. Select the Overview... section to expand it's options.
  3. In the Asset Class drop down, select a Custom Asset Class to add to the asset.

    4. This adds an additional field below called Custom fields....

  4. Select the Custom fields... option to view the custom asset fields.
  5. Enter the necessary information and select Save Asset when finished.

The new Custom Asset Class field and its associated information are now available for the selected customer's asset in the Assets section of your account.
