Export your invoices as a CSV file

If you are not using QuickBooks Desktop, QuickBooks Online, or Xero, you have the option to export your invoices as a CSV file.

  1. Navigate to Billing, and run a billing batch.
  2. Select each invoice you would like to export, and choose CSV File or Raw CSV File as the export format.

  3. Click Export Invoices to generate the CSV file.
