Remove the web connector file from QuickBooks Desktop

QuickBooks web connector errors can interfere with and prevent the export of invoices from to QuickBooks Desktop. Removing and reloading the web connector file is a good way to diagnose and fix any errors that occur while exporting invoices.

  1. Open QuickBooks Desktop and select Edit > Preferences.
  2. Select Integrated Applications from the options on the left.
  3. Select the MSP Manager service and click Remove.

  4. Open and select the Billing tab.
  5. Create a new billing batch or select an existing batch.
  6. Select the Review and Export Invoices tab.
  7. Select QuickBooks Desktop from the Export Format drop down menu.
  8. Click the Downloaded QWC File link to download the QuickBooks Web Connector file.

After the download is complete, the web connector will relaunch and ask for permission to re-establish the connection with . Complete the set up process and run the web connector to import your invoices into QuickBooks.
