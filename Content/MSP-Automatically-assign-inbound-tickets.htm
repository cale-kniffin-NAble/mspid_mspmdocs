Configure ticket routing

allows ticket creation from external sources, including by email, the customer portal, and from N-central.

To view inbound requests, click Helpdesk > Ticket Requests. Click Create Ticket to convert an inbound request to a new ticket.

The ticket routing feature allows an externally created ticket request from the customer portal or via email to be automatically converted to a new ticket, and assigned to a customer and technician.

How does ticket routing work?

uses ticket routing rules to determine which users to assign to a ticket, as well as an algorithm to evaluate each user against the ticket routing rules.

Each ticket routing rule produces a value in the range of 0 - 5 for each user. The user with the highest score is assigned the ticket. In the case of a numerical tie, assigns the ticket to all of the users with the highest score.
