Description

  • Error when opening ticket in through

Environment

Solution

  1. Log into
  2. Go to Settings > Ticket Routing > Ticket Routing Rules > When a ticket comes in via RMM...
  3. Ensure Only create tickets when the Asset class is: Make sure this shows Server and Workstation
  4. Click Save Settings
