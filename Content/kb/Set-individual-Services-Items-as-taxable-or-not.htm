Set individual Services Items as taxable or not

Last Modified

Tue Apr 23 14:17 GMT 2019

Description

  • How to set individual Services Items as taxable or not
  • For example, when I repair a computer the labor is not taxable, but when I install a new computer the labor is taxable.

Environment

Solution

Set service items to default to taxable on all tickets. Any tickets that are opened using this service item are marked as taxable.

  1. Click Customers left panel, then select the desired customer from the list.
  2. Click Service Items.
  3. Click the icon
    next to a service item.
  4. Under Billing > Tax Default , select Default All New Tickets to Taxable box.
  5. Click Save Changes top right corner.

Expenses that are tied to a taxable ticket can be marked as taxable or not, separate from the service item this when the expense will be added to a ticket.
