IMAP Connectivity test for ticket mailbox

Last Modified

Tue Jul 02 11:59 GMT 2019

Description

  • This article provides information on how to perform IMAP connectivity test for ticket maiboxes

Environment

  • Microsoft IMAP Test connectivity

Solution

  • When troubleshooting Ticket Mail box for , to prove that partners mailbox is IMAP enabled we recommend them to run an IMAP Connectivity test from Microsoft.
  • https://testconnectivity.microsoft.com/

  • Note: Normally if they get a successful connection but in the status is still not ok, have them re-add the mailbox and if problem persist create an SI following the template
