Can you edit a ticket if status is already Billed?

Last Modified

Thu Dec 13 12:09 GMT 2018

Description

  • Can you edit a ticket if status is already Billed?

Environment

Solution

  • Billed tickets cannot be edited. They would need to be removed from the billing batch that they were invoiced from.
  • To delete the Billing batch please follow the steps below:
  1. Login to
  2. Click Billing
  3. Select the Billing batch name
  4. Click Delete (this will delete the whole billing batch and all tickets associated with this will change it's status to "Review")
Note: You can also remove a specific ticket without deleting the whole billing batch in case you just want to edit single or multiple tickets but not all the tickets associated with the billing batch.
