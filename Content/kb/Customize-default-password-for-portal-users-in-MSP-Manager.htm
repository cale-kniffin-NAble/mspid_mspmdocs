Description

  • When creating a new Customer contact with access to portal what is the default passoword?
  • Can this default password be customizable?

Environment

Solution

  • When creating a new Customer contact with access to portal the user will be emailed an invite with a unique activation code
  • After the Customer contact logs into the portal they are able to change their password
  • There is no way to customize the password that is sent to customer
