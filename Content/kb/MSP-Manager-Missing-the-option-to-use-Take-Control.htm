: Missing the option to use

Last Modified

Wed Dec 18 13:21 GMT 2019

Description

  • Trying to use from in the Assets tab
  • button is missing

Environment

Solution

  • This option becomes available for assets that have been synced over from RMM integration
  • Log in to app.mspmanager.com and click on the Assets tab
    • Under the Actions Column there will be a take control option for devices that have enabled in RMM and have been synced over from integration
