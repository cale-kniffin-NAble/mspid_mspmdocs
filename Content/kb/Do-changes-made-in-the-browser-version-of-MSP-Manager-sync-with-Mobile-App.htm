Do changes made in the browser version of sync with Mobile App?

Last Modified

Fri Oct 26 20:14 GMT 2018

Description

  • Do setting configuration changes sync to mobile app after changing them on desktop GUI?

Environment

Solution

  • Mobile Agent syncs to dashboard automatically and reflects changes made in desktop user interface
  • Mobile App may need to be refreshed before configuration changes are reflected in App
  • To refresh Mobile App
    1. Completely close down Mobile App
    2. Reopen App
    3. Login
