Workflow Rules are not applying to some customers, or not applying at all

Last Modified

Fri Mar 20 08:09 GMT 2020

Description

  • One or more Workflow Rules have been created
  • The Rules are working for some customers, OR they are not working for any customers

Environment

  • Workflows and Rules

Solution

  1. Navigate to Settings > Tickets & Workflows > Workflow Access Groups
  2. Make sure there is a Group added for the Event Type the Rule is under
    • If you are not sure what Event Type the Rule is under:
      1. Navigate to Settings > Tickets & Workflows > Workflow & Rules
      2. Open each category to find the one with the Rule that is not applying as expected
  • If there is a rule added for this Event Type, click the Edit icon
    and either assign the rule to All Customers using the radio toggle, or select the customers in the Customers dropdown list
  • If there is not a rule added for this Event Type, click the Add button, configure the rule for the Workflow Event Type and either assign to All Customers or choose the customers in the Customers dropdown list
