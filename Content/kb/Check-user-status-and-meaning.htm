Check user status and meaning

Last Modified

Wed Nov 30 21:47 GMT 2016

Description

This article details where you can verify all the users in and the meaning of those statuses.

Environment

All versions of

Solution

You can verify the User status on by going to Account Settings > User Management.

Different kinds of User Status:

  • Waiting for Activation - Upon initial creating, this would send an activation email to the configured user's email address, once verified, the status will change to Active.
  • Active - User account is verified and allowed access as specified by its Role.
  • Disabled - User account disabled
