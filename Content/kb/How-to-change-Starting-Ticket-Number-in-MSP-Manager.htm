Description

  • How to change Starting Ticket Number in ?
  • Customers want to continue off from their old ticketing systems ticket number
  • This article explains how to set the ticket number at something other than ticket #1

Environment

Solution

  1. Log into
  2. Click on Settings > General
  3. Locate the 'Next Ticket Number' field
    • Note: Allows you to change the next ticket number that will be generated. Your next ticket number cannot be smaller than your current latest ticket number
  4. Change the ticket number
  5. Click Save Settings
