SQL timeout when creating new billing batch in

Last Modified

Mon Oct 22 14:23 GMT 2018

Description

  • SQL timeout when creating new billing batch in

Environment

Solution

  • This is caused by complex queries running on the database
  • This is often the cause when running high number of invoices in one billing batch
  • To minimize this from happening reduce the amount of invoices in the billing batch
  • There are no restrictions as to how many you can run at once, but if you are getting this error try reducing the amount by half and reduce/increase from here
  • If this fails to help, open a technical support case.
