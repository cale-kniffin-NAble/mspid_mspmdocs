Unable to view ? (Euro) symbol in Excel when viewing a .CSV export

Last Modified

Fri Oct 19 19:48 GMT 2018

Description

  • When opening an billing CSV export in Excel, the ? (Euro) symbol/character does not display correctly

Environment

  • Microsoft Excel

Solution

  • This is caused by Microsoft Excel being unable to properly display UTF-8 compliant CSV files when they contain non-English characters (In this case it is considering the ? to be a non-English character)
  • To fix this issue please follow these step:
  1. Open the CSV file using Notepad
  2. Click File > Save As
  3. In the dialog window that appears, select "ANSI" from the "Encoding" field. Then click "Save"
  4. Open this new CSV file using Excel and the non-English characters should display properly
