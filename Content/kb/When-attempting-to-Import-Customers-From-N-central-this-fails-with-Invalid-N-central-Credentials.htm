When attempting to "Import Customers From ", this fails with "Invalid Credentials"

Last Modified

Wed May 08 10:01 GMT 2019

Description

  • Under Settings > Company Settings > Integrations > N-central when attempting to import customers the following error occurs:
    • Error:
  • Opening Developer Tools > Network shows a failure connecting to https://api.mspmanager.com/api/NcentralCustomer with the following error:
    • Invalid Credentials

Environment

Solution

  • This is either due to the credentials being used are incorrect or the URL is invalid
  • To resolve please do the following:
    1. Please update the credentials to ensure the username and password credentials are correct
    2. Ensure that under Settings > Company Settings > Integrations > N-central in the Credentials section that the URL has the appropriate 'http://' or 'https://' prefix
