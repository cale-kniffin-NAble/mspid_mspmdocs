Deleting users in causes them to be deleted in

Last Modified

Thu Oct 10 19:53 GMT 2019

Description

  • Deleting users in causes them to be deleted in
  • is integrated with

Environment

Solution

  • Integrating with causes all data to be synced between the two products.
  • Users created in will be synced to .
  • Deleting a user in will delete them in .
  • If RMM users are not going to be used in , a user should be disabled rather than deleted. To disable:
    1. Open
    2. Click on Settings
    3. Select Users & Permissions
    4. Select Users
    5. Select the pencil icon on right of the desired user and select Edit
    6. Under Status, uncheck the box to the left of "Enabled"
    7. Select Save Changes ("save" icon on the top right).
