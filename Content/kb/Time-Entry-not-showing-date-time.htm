Time Entry not showing date/time

Last Modified

Fri Oct 26 19:25 GMT 2018

Description

  • When adding a time entry to a ticket in NGP, the Start Time and End Time fields show "un10e0ine10 un10e0ine10"
  • Still allows to input date and time by clicking the buttons
  • Difficulty reading data

Environment

Solution

  • Attempt the following:
    1. Clear your browser cache
    2. Clear your cookies
    3. Close browser
    4. Reopen browser
    5. Log in
    6. Attempt to add time entry
      • Attempt on different browser to see if it is browser specific
      • Attempt on different device to see if it is device specific
