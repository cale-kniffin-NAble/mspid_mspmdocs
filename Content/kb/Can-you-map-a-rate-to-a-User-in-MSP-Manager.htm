Can you map a rate to a User in ?

Last Modified

Fri Oct 26 12:53 GMT 2018

Description

  • Have two (2) rates for a User
  • Can you map a rate to a User in ?

Environment

Solution

  • Rates cannot be mapped directly to a User in
  • Create a Service Item and then create Rates under that Service Item
  • Create a ticket with that Service Item and rates will be available for tech to select from
