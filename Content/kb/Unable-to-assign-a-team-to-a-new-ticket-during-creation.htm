Unable to assign a team to a new ticket during creation

Last Modified

Sun Mar 01 12:45 GMT 2020

Description

  • A team has been created via the Settings > Users & Permissions > Teams option within
  • Created team is unable to be assigned to a new ticket during the creation phase
  • Users can only be added one by one

Environment

Solution

  • This is expected behaviour.
  • To add groups of users to any new tickets, you would use the Queues Feature for this
  • Queues can be found under Settings > Tickets & Workflows > Queues
  • From here, you can name the Queue, add a description and assign all relevant users to that queue
  • This will then allow you to add the queue of users during ticket creation done manually or via Workflow & Rules
