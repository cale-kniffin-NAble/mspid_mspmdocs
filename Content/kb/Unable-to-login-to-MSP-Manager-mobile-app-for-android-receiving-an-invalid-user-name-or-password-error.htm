Unable to login to mobile app for android, receiving an invalid user name or password error

Last Modified

Fri Oct 19 19:51 GMT 2018

Description

  • Unable to login to mobile app for android, receiving an invalid user name or password error

Environment

  • mobile
  • Android devices

Solution

  1. Verify the account exists and is active under account settings > User Management
  2. When entering the account information ensure there are no extra spaces before or after the Username or Password
  3. When entering the Username do not use auto-complete ensure you are typing out the entire address
  4. The username is case sensitive watch to see if characters are being capitalized incorrectly
  5. Test with another account, does it work?
    1. If yes then its likely either the account that isn't working, is inactive or something extra is being added try again to retype it
  6. Try the account on another device if available, does it work?
  7. If it is still not working we will need a case logged and escalated to the Application Engineers (In the case we will need the OS version and hardware version)
