Exporting to Quickbooks shows the error "QuickBooks error message: The specified name is either invalid or of the wrong type."

Last Modified

Mon Oct 22 15:03 GMT 2018

Description

  • Exporting to Quickbooks shows the error "QuickBooks error message: The specified name is either invalid or of the wrong type."

Environment

Solution

  • Need to change customer type in Quick Books
1. In QuickBooks' top menu, click lists > other names list
2. Locate the customer record
3. Double click the row
4. Click the change type button:
5. Select Customer
6. When promoted that the change cannot be undone, click OK
