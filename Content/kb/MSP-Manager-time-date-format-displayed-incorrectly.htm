time/date format displayed incorrectly

Last Modified

Mon Oct 22 19:41 GMT 2018

Description

  • Date format was DDMMYYYY it has now reverted to MMDDYYYY
  • Culture setting is correctly set to correct country under Settings > Company Settings > Default Settings > Culture

Environment

Solution

  • Currently the Culture settings is being overwritten by browser settings
  • There is plan to change it in the future so browser settings do not impact way of displaying pages in the app
  • In Firefox:
    • Options > Language > Choose your preferred language for displaying pages
    • Add <language> to the top of languages list
    • Restart browser
  • In Chrome:
    • Settings > Language > Order languages based on your preference
    • Add <language> to the top of languages list
    • Restart browser (if does not help then set language as Chrome displayed language)
