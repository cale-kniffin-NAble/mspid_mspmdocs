How to apply Service Plan to multiple customers

Last Modified

Mon May 20 10:26 GMT 2019

Description

  • How to apply Service Plan to multiple customers

Environment

Solution

  1. In the dashboard, go to Settings > Billing & Financial >Service Plans
  2. Click Add Service Plan
  3. Select the Service type, enter a name and the billing, rates, and invoicing information
  4. Click SAVE SERVICE PLAN
  5. Select Default Service Item
    • NOTE: Before applying a Service Plan you must add service plan as a default or it will not allow editing
  6. In the list of Service Plans, click to apply the plan to multiple customers
  7. Select the customers you would like to apply the service plan to
  8. Click Assign Service Plan
