Does have an option to make all tickets non-billable?

Last Modified

Wed Oct 31 10:06 GMT 2018

Description

  • Does have an option to make all tickets non-billable?
  • Cannot find global option to make all new and opened tickets non-billable

Environment

Solution

  • Currently at this time there is no global option to make tickets non-billable
  • You can deselect the billable option during the creation of a ticket or edit ticket to deselect the billable option
  • Please vote on this feature request at http://ideas.n-able.com/a/dtd/Global-billable-non-billable-option-for-tickets/302459-4178?submitted=1#idea-tab-comments
