Add Location or Contact Permission

Last Modified

Thu Aug 08 14:22 GMT 2019

Description

  • How do I add a Contact in ?
  • Add Location/Contact button is missing in Customer > Contact information

Environment

Solution

  • There are two causes for this issue
    • The User doesn't have appropriate permissions to add a Contact
    • There is no Location listed under the Customer
      • A Contact cannot be added unless there is a Location listed for that Customer
      • Select the Customer > Contact Info > Add Location
  • To fix permissions issues:
  1. Go to Account Settings > Roles
  2. Click the Edit button (pencil) for the Role for the User experiencing the issue
    • To check which Role is Assigned to the User, go to Account Settings > Roles
  3. Click Customer Management section
  4. Click the checkbox for Add/Edit Customer Locations and Contacts
  5. Click Save
