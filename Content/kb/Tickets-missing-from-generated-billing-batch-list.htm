Tickets missing from generated billing batch list

Last Modified

Tue Mar 12 21:32 GMT 2019

Description

  • Billing doesn't reflect every ticket ready to be billed

Environment

Solution

  • Billing batch filter only works with the last service item range
  • Specify filter to work with tickets that were completed earlier or later than the current range
  • Prior to version 3.0, tickets must be marked Complete to appear ready for billing
  • In versions 3.0 and later, partial billing has been added for select service item types (flat fee, hourly, etc.) to allow tickets to be billed that are not marked complete
  1. Open the problem ticket.
  2. Take note of the completion date.
  3. Create a new billing batch:
    • Billing > ADD BILLING BATCH click Filter Service Items to open Filter menu.
  4. Under Ticket Completed Date Range, set starting day and ending day that includes completion date of ticket
  5. Verify billing period also includes ticket date before completion.
