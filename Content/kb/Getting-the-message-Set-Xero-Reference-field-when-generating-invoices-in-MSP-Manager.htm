Getting the message "Set Xero Reference field" when generating invoices in

Last Modified

Tue Oct 23 13:56 GMT 2018

Description

  • Getting the message "Set Xero Reference field" when generating invoices in
  • When exporting an invoice from to Xero the Reference ID is empty

Environment

  • Xero

Solution

  • The Reference ID needs to be added for customer before invoice is exported
  • To add name go to Customers > select the customer > Edit Customer > under Advanced at the bottom enter "Custom reference ID"
