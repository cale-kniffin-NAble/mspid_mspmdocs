When trying to create a ticket through integration, generating a 10606 error "No default service item has been configured for this customer"

Last Modified

Tue Sep 01 18:00 GMT 2020

Description

  • When trying to generate a ticket from a device in to an integrated , getting "error 10606: No default service item has been configured for this customer"
  • Getting the following error message when trying to create a ticket:
    • Error: 10606 Unable to create a ticket, please contact N-able Technical Support for assistance.
    • The following is the error message provided by :
    • No default service item has been configured for this customer.
    • Please confirm that there are no networking issues, and that the Ticketing Recipient is configured properly

Environment

Solution

  • This issue is due to no default service item being applied to the customer
  • This can either be resolved from the side or
  • It is recommended to do this from within the side at Account level as this would apply to all customers, so it does not need to be applied on a per customer basis
    • From within (Account Setting level):
      1. Select Settings
      2. Select Account Settings
      3. Select Service Plans
      4. Under the Actions section, select Edit for the Service Plan you would like to be set as default
      5. Check the option for Default Service Item
      6. Select Save Service Plan
    • From within (per customer basis):
      1. Select Customers
      2. Select the customer in question
      3. Select Service Item
      4. Under the Actions section, select Edit for the Service Item you would like to be set as default
      5. Check the option for Default Service Item
      6. Select Save
    • From within :
      1. Select Administration
      2. Select PSA Integration
      3. Select Ticketing
      4. Select the desired Ticketing Recipient
      5. Select the Recipient Details tab
      6. Select Add
      7. Select the customer in question from the Customer dropdown
      8. Select the desired Service Item
      9. Select Save
