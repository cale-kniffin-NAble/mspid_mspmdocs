: Cannot Change Existing Help Desk Ticket To Another Customer After Adding Time entries

Last Modified

Tue May 15 17:03 GMT 2018

Description

  • Cannot edit an existing help desk ticket and change from one client to another after adding time entries

Environment

Solution

  • This is by design
  • Only time a customer can be changed on a ticket is when creating the ticket, or if the ticket was created and no time entries were added
  • Once a ticket has a time entry added you would need to delete the ticket and create a new one for the correct client
