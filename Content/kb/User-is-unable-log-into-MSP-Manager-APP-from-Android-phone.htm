User is unable log into APP from Android phone

Last Modified

Mon Oct 22 17:54 GMT 2018

Description

  • User is unable log into APP from Android phone

Environment

Solution

  1. Quit Application
  2. Clear cache on the phone (will vary by model)
    • Navigate to Settings > Storage > Cache data > Clear
  3. Launch Application
  4. Attempt to Login
