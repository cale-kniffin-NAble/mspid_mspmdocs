ERROR: 'Something went wrong. Xero was unable to create the Invoice. Check Account Codes and try again' when exporting invoices to Xero

Last Modified

Tue Mar 19 12:32 GMT 2019

Description

  • Xero is integrated with correctly
  • The ticket on the invoice that is being exported is billable
  • Codes in and Xero are correct and match
  • ERROR: Something went wrong. Xero was unable to create the Invoice. Check Account Codes and try again

Environment

Solution

  • This error indicates that the codes in are not aligned with Xero
  • To ensure both platforms match. In :
  1. Navigate to Settings
  2. Select Integrations
  3. Select Configure under the Xero banner
  4. Select Import from Xero
  5. Run through the integration windows (Unblock popup blocker)
  6. Set Mappings
  7. Click Save Default Mappings
  8. Attempt to export a billing batch to Xero
