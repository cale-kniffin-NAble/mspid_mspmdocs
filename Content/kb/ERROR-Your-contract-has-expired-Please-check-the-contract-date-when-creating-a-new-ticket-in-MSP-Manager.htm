Description

  • ERROR: "Your contract has expired. Please check the contract date" when creating a new ticket in

Environment

Solution

  • This message will normally display due to the service item date that you set
  1. Go to Customers
  2. Select the customer
  3. Click Service items
  4. Click Edit button for Service item
  5. Adjust the end date
  6. Click Save
