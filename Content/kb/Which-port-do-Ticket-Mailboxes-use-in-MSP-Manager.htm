Which port do Ticket Mailboxes use in ?

Last Modified

Mon Jul 01 14:02 GMT 2019

Description

  • Which port does do Ticket Mailboxes use in ?

Environment

Solution

  • Ticket Mailboxes can only monitor IMAP mailboxes ( port 143 or 993 for SSL)
  • Port 25 is for SMTP and cannot be used
