Will Users integrated from RMM into be chargeable?

Last Modified

Wed Jan 24 17:24 GMT 2018

Description

  • RMM users have been integrated into
  • users have been created
  • I don't want to have all users in RMM as active users in
  • Will I be charged for these users?

Environment

  • Users
  • Billing

Solution

  • If you make sure the Users in you wish not to be billed are not Enabled, you will not be charged
  1. Open and log in
  2. Click on the Cog Icon > Account Settings
  3. Click on Users
  4. Click on the Edit icon next to the specific user
  5. Make sure the Enabled box is unticked
  6. Tick I acknowledge I have read this and select Save Changes
