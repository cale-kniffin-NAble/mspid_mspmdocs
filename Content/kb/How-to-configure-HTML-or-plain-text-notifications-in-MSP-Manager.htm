Last Modified

Description

  • How to configure HTML or plain text notifications to be send to the customers and technicians?

Environment

Solution

  • Create notification templates to alert your customers or technicians about updates to tickets, or changes to knowledge and service items. Choose from plain text or HTML notifications to keep your customers in the loop anytime information is added to a ticket. Use the Data Model to understand how to use each variable in the template:
  1. You must enable the template in order to send the notification. Changes to the template are saved as a draft and will not be sent until enabled.
  2. Click Settings > Notifications > Notifications Templates
  3. Select a template from the list to expand additional options
  4. Select the External tab to configure notifications that are sent to your customers, or the Internal tab to configure the notifications sent to your technicians
  5. Choose a language and select a format.
  6. Click the Pencil icon to edit and customise the template
  7. Click Arrow (top right) to open the Data Model and view information about each variable in the template
  8. Select Actions > Validate to verify your changes are valid
  9. Click Preview to view the notification
  10. Check Enable Custom Template to activate the template and make it ready to be sent
  11. Click Save
  12. Specify the template you would like to apply to each notification
  13. Click Save

Internal notes: Only users with Administrator level permissions can configure notifications.
