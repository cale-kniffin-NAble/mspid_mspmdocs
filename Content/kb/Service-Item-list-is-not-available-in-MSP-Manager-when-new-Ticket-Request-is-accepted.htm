Service Item list is not available in when new Ticket Request is accepted.

Last Modified

Tue Aug 20 18:35 GMT 2019

Description

  • Service Item list is not available to any user in when new Ticket Request is accepted

Environment

Solution

  • Make sure Service Plan is set to Active:
  1. Log into
  2. Go to Settings > Billing > Service Plans
  3. Click Edit Service Plan for Service Plan in question
  4. Make sure Status Active box is checked
  5. Click on Save Service Plan
