How to see the available variables for editing notification templates

Last Modified

Fri Apr 10 12:57 GMT 2020

Description

  • How to see the available variables for editing notification templates

Environment

  • Notification Templates

Solution

  • To see the available variables for editing notification templates:
    1. Select Settings
    2. Under the Notifications, select Notification Templates
    3. Select the template that you would like to edit
    4. For the relevant template to change, click on the pencil icon to edit
    5. Click the expand icon < which is next to the save button
    6. Doing so should then expand the Available Variables from Data Model table
