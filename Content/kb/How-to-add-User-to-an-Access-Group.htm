How to add User to an Access Group?

Last Modified

Wed Sep 02 01:47 GMT 2020

Description

  • Procedure how to add User to an Access Group in

Environment

Solution

  1. Click Settings > User & Permissions > Access Groups > Add Access Group.
  2. Enter a name and description, and click Enabled.
  3. Select one or more assigned customers.
  4. Select one or more assigned users.
  5. Save the Access Group.
