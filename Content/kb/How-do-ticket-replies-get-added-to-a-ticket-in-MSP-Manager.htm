How do ticket replies get added to a ticket in

Last Modified

Tue Jul 04 14:03 GMT 2017

Description

  • This article provides information when end customers receive an email from and they want to be able to reply to the email, and have it added to the ticket.

Environment

  • All versions of

Solution

  • When an email is sent out from a ticket in it includes a "TicketID[xxxxxx]" at the bottom of the email.
  • If the customer replies to the email from and they leave the "TicketID[xxxxxx]" in the email then should pick it up and append it to the existing ticket.
    • Note: This will not append to a Billed ticket. If the ticket is already Billed will open a new ticket.
  • This is an intentional design, set up so that it does not interfere with your billing.
