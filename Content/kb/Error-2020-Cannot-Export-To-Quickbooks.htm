Error: 2020 Cannot Export To Quickbooks

Last Modified

Thu Mar 07 14:17 GMT 2019

Description

  • When exporting an invoice to quickbooks online / receive the following error:
  • "Sorry. Something went wrong. Problem While adding BilledInvoiceItems: ErrorCode :2020 Exception : Required param missing, need to supply the required value for the API Details: Required parameter Line is missing in the request"

Environment

Solution

  • This problem occurs when there is no value entered in the Line Items field
  • Delete the invoice or edit the invoice that is missing the line item entry and then re-export

Article Number: 110388
