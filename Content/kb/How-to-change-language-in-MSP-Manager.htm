Description

  • How to change language in ?

Environment

Solution

  • When logged in under app.mspmanager.com you will be unable to change the UI language
  • To change culture language please do the following:
  1. Click on Gear Icon to far right
  2. Click Account Settings
  3. Under Company Settings click Default Settings
  4. Under Culture drop down select culture you want to apply to your account, This will change the date format but not the UI
  5. Click Save Settings
  • Note: To change the UI language, please contact Product Management to get access to - NGP Beta version
  • When logged in under ngp.mspmanager.com you can change the UI language by doing the following:
  1. Log into ngp.mspmanager.com
  2. Click on your name at top right
  3. Select Language
  4. Click language to switch over to
