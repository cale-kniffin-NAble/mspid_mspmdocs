Description

  • How to resolve notifications which are being blocked as spam/junk mail

Environment

Solution

  • Add a rule in the email system that in use to make sure all emails are received for:
    • mspmanager.com
  • Depending how your mail domain is configured you may need to create a SPF record. For more information, see How to set up Sender Policy Framework (SPF) for

Article Number: 125719

Internal notes: If it still gets classified as spam even after adding the SPF Record, Sender Authentication may have to be enabled. Please obtain the email header and the domain they should be sending from and collaborate with Application Engineer to raise an NCI for this request.
