How to add multiple domains to a customer in

Last Modified

Sun Mar 15 21:35 GMT 2020

Description

  • How to add multiple domains to a customer in
  • A customer uses multiple email domains

Environment

Solution

  1. Select the Customer that needs the extra domains added
  2. On the Dashboard tab, click the Actions dropdown at the top right
  3. Select Edit Customer
  4. Insert the domains into the Email domain field
  • NOTE: Each domain should be separated by a comma
