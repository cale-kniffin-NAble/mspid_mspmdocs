Where to set the Tax rate for tickets in

Last Modified

Tue Mar 21 09:26 GMT 2017

Description

  • Where to set the Tax rate for tickets in

Environment

  • Account Settings

Solution

  1. Open up
  2. Click on the Cog icon in the top right hand corner and click Account Settings
  3. Then click Default Settings
  4. Modify the Tax Rate value (in percent)
  5. Click Save Settings
