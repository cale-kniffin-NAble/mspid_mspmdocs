Description

  • Steps on how to change user role in

Environment

Solution

  • To change the role of a user follow the steps below:
  1. Login to
  2. Go to Settings
  3. Select Users & Permissions
  4. Click Users
  5. Search for the user and click the pencil icon under Action to Edit User
  6. Under Role, select the role that you want to apply
  7. At the bottom, tick the box beside I acknowledge I have read this.
  8. Click the icon to Save Changes on the upper right.
