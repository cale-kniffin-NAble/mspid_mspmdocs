Ticket routing with multiple locations

Last Modified

Wed Nov 30 21:47 GMT 2016

Description

I configured email ticket routing by location. The email domain of the sender matches a customer with multiple locations, but the sender is not a contact for that customer. How is the ticket routed?

Environment

Solution

selects the default location of the customer. If a default location is not selected, selects the first location.
