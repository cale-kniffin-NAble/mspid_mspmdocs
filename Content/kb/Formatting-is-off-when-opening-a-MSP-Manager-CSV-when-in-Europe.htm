Formatting is off when opening a CSV when in Europe

Last Modified

Mon Oct 22 12:24 GMT 2018

Description

  • What to do after opening the Billing export CSV file in a European version of Excel and the formatting is incorrect

Environment

Solution

  • This issue is related to how Excel functions in North America as opposed to Europe
    • In North America a CSV file uses commas to separate variables
    • In Europe a CSV file uses semicolons to separate variables

  1. Download CSV and save it locally
  2. Open CSV file with a text editor (like Notepad ++)
  3. Add a New line to beginning of file and enter: sep=,
  4. Save file
  5. Re-open file in Excel
  6. Formatting should now be correct

  • NOTE: If it is still incorrect please log a case with support and include the name of the billing batch, a copy of the CSV and a screenshot of how it appears for you.
