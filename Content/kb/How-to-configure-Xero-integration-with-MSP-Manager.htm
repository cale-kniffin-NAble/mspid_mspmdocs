Environment

Solution

  • To import your Xero Account into , please do the following:
  1. Disable the pop-up blocker in your browser
  2. In , select Account Settings > Integrations > Xero > Configure
  3. Select Import from Xero
  4. Select OK to confirm you disabled the pop-up blocker
  5. Log in with your Xero account
  6. On the Authorize Application window, select Authorize
  7. Select Import from Xero
  8. Select the inventory items you want to import into
  9. Select Import
  10. Select Close
  11. Under Default Mappings, map each line item to the defaults listed
  12. Select Save Default Mappings
