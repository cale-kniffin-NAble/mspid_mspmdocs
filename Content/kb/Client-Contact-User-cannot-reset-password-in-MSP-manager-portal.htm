Client Contact User cannot reset password in portal

Last Modified

Mon Apr 29 14:41 GMT 2019

Description

  • When you click the reset password link in the portal and enter the email address of the portal user it shows an error: "Error accessing shardmap:MspManagerShardMap_Contact with key..."

Environment

Solution

  • Check if the contact exist in the
    1. Login to
    2. Click Reports
    3. Select Customer Reports
    4. Click Location and Contacts
    5. Select All Customers and check the include contacts
    6. Click Apply filter
    7. Check if the email address exist as one of the contact
  • If the contact exist, verify if its set as a portal user/admin
    1. Login to
    2. Click Customer
    3. Select the customer name where the contact is added
    4. Click on contact and edit that contact
    5. Make sure that portal user box under customer portal is ticked
