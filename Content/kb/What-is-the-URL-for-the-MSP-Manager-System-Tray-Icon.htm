What is the URL for the System Tray Icon?

Last Modified

Fri Jul 20 19:03 GMT 2018

Description

  • What is the URL to which the System Tray Icon links?

Environment

    • System Tray Icon

Solution

  • System Tray Icon links to Customer Support Portal URL https://portal.mspmanager.com
  • If using branded customer portal, URL is https://CUSTOMERBRANDEDNAME.portal.mspmanager.com
