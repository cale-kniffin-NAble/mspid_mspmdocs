What are the settings for an Office 365 ticket mailbox account?

Last Modified

Mon Jun 15 13:05 GMT 2020

Description

  • What are the settings for an Office 365 ticket mailbox account?

Environment

  • Ticket Mailboxes

Solution

  • When setting up a ticket mailbox for the only configuration details you need to set in are as follows:
    • Email Address, Server Address, Server Port, Username and Password
    • Note: Username and Password would be the username and password of the admin account for Office 365. The host will need to be outlook.office365.com and the user must be valid.
    • Note: Server settings for Outlook.com or Microsoft 365 for business email accounts here
  • To set up a ticket mailbox in
    1. Use testconnectivity.microsoft.com to verify correct setting for the mailbox
      • Note: Microsoft has enforced Security Defaults for new O365 Tenants, this will disable IMAP.
      • is aware of this. For more information see, Ticket Mailbox not connecting to Office 365 due to MFA requirement
    2. Click Settings > Tickets & Workflows > Ticket Mailboxes > Add Mailbox
    3. Enter information for ticket mailbox in
    4. Click Save Mailbox
  • Note: Office 365 shared mailboxes are not supported. A feature request has been submitted here
