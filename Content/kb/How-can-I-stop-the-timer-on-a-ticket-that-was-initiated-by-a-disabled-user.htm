How can I stop the timer on a ticket that was initiated by a disabled user?

Last Modified

Wed Jul 03 12:43 GMT 2019

Description

  • How to stop the timer on a ticket that was started by an inactive tech?
  • The ticket cannot be marked as complete due to the active timer
  • ERROR: 'This ticket cannot be closed because another user has an active timer running on this ticket.'

Environment

Solution

  • Only the Company Owner is able to login and stop the timer of any user in the system
  • This is primary user account which initially setup

Article Number: 108537

Internal notes: Super Users can check this in the Utility App > Company Owner
