Error when logging into Customer User Portal for

Last Modified

Mon Oct 22 12:53 GMT 2018

Description

  • Error when logging in to Customer User Portal
  • Complete error message below:
    • Culture is not supported. Parameter name: name null is invalid culture identifier

Environment

Solution

  • Issue happens when incorrect details (Timezone and Culture) are configured in the Default Settings
  • To resolve this issue, have the correct details configured in Settings > Default Settings > TimeZone/Culture
