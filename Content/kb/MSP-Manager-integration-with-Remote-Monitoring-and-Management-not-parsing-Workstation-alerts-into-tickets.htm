integration with RMM not parsing Workstation alerts into tickets

Last Modified

Wed Aug 19 08:55 GMT 2020

Description

  • RMM Workstation alerts are not creating tickets in
  • By default only Server alerts are configured

Environment

Solution

  1. Within go to Account Settings (Cog top right)
  2. Go to Ticket Routing Rules
  3. Click on When a ticket comes in via RMM...
  4. In Only create tickets when the Asset class is: type in Workstations
  5. Configure settings to preference
  6. Click on Save Settings
