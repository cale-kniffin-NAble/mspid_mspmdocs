How to disable a user in ?

Last Modified

Fri Aug 28 23:46 GMT 2020

Description

  • User would like to disable a user/ technician in ?

Environment

Solution

  1. In , go to Settings
  2. Click on Users & Permissions
  3. Click on Users
  4. Beside the user you wish to deactivate click the Pencil icon (Edit User)
  5. Under Status, uncheck the Enabled box
  6. Click on the Save icon (Save Changes) on top
