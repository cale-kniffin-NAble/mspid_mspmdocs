How to include assigned users in the notification template

Last Modified

Mon Jan 28 20:11 GMT 2019

Description

  • How to include assigned users in the notification template

Environment

  • Notification Template

Solution

  • To include the assigned users variable in the notification template, please do the following:
  1. Select Settings
  2. Select Account Settings
  3. Under the Templates option, select Notifications
  4. Select the ticket template in question you would like to edit
  5. For the relevant template you wish to change, select on edit under the actions column
    • NOTE: Only users with Administrator level permissions can configure notifications
    • If this is still not available, please raise a case with technical support who will be able to get you access
  6. Input the following into the desired location in the template:
    • {{ Model.AssignedUsers | Select: '(item.FirstName) (item.LastName) (item.EmailAddress)' | Join: ', ' }}
      • What this does is it inputs the assigned user's first and last name, as well as their email address with a comma separating them
        • NOTE: To see the list of available variables, please view this article: How to see the available variables for editing notification templates
  7. Once you have done this, select Preview to see if it is displayed as you would like
    • NOTE: If this was entered correctly, then the preview should show the test values, which can be found by following the mentioned article in step 6
  • For more information on this, please go here
