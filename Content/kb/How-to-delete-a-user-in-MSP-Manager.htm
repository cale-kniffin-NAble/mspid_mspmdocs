How to delete a user in

Last Modified

Mon Jun 01 08:23 GMT 2020

Description

  • How to delete a user in standalone or when integrated with ?

Environment

Solution

  • In most cases, a User should be disabled rather than deleted. To disable:
    1. Open
    2. Click on Settings
    3. Under User Management, click on Users
    4. Edit the desired user
    5. Uncheck Enabled
    6. Click on Save Changes
  • To delete:
    1. Open
    2. Click on Settings
    3. Under User Management, click on Users
    4. Beside the user you wish to delete, click the Trash icon on the right side of the user
    5. Click OK
  • NOTE: User which have Time Entries cannot be deleted
  • If the user was created from integration with RMM, change the email address of the user in :
    1. Go to Settings
    2. Under Users Management, select Users
    3. Sync the changes with and then disable the user
