ERROR: 'Make sure that you have included all required fields in your request.' when creating a ticket in

Last Modified

Wed Apr 24 08:20 GMT 2019

Description

  • When trying to save a ticket request in
  • ERROR: 'Make sure that you have included all required fields in your request.'

Environment

Solution

  • The following fields are mandatory when saving a ticket request:
    • Customer
    • Service Item
    • Status
    • Priority
    • Ticket Title
