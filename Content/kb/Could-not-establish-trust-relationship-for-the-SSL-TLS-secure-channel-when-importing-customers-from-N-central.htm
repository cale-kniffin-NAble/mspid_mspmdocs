When attempting to "import customers from " getting "Error: The underlying connection was closed: Could not establish trust relationship for the SSL/TLS secure channel."

Last Modified

Mon Nov 30 22:15 GMT 2020

Description

  • Under 'Settings' > 'Company Settings' > 'Integrations' > 'N-central' when attempting to import customers the following error occurs:
    • Error: The underlying connection was closed: Could not establish trust relationship for the SSL/TLS secure channel
  • Getting a '404 unauthorised' when attempting to import customers from , via https://api.mspmanager.com/api/NcentralCustomer

Environment

Solution

  • This is due to the certificate on the server not using a fully certified SSL certificate
  • To resolve please generate a CSR and obtain a valid SSL certificate
  • More information on this can be found here:
    • Create a CSR and install SSL certificate for the server
    • SSL for - Chaining your certificate
