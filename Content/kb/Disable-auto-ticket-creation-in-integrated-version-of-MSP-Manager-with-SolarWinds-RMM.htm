Disable auto ticket creation in integrated version of with

Last Modified

Wed Jan 02 10:27 GMT 2019

Description

  • How to disable auto ticket creation in integrated version of with

Environment

Solution

  1. Log into console.
  2. Navigate to Settings.
  3. Select Account Settings.
  4. Under TICKET ROUTING, click on TICKET ROUTING RULES.
  5. Select When a ticket comes in via RMM...
  6. Configure ticket creation as per your requirements: Configure ticket routing
  7. Click on SAVE SETTINGS.
