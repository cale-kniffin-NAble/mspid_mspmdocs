Can a request be placed to remove a Password Secret in ?

Last Modified

Thu Mar 14 15:24 GMT 2019

Description

  • Set a password secret
  • Other Tech's do not know what the password secret is
  • Can a request be placed to remove a Password Secret in ?

Environment

Solution

  • There is no way to remove just the secret for an individual password while at the same time retaining the password. We cannot then show the password in plain text because that password is 'encrypted' in a way that it is not recoverable by us for security reasons.
  • A company can use an Owner Secret and set it. This can be used to record a password with a forgotten secret. The Owner Secret is used to secure all future passwords.
