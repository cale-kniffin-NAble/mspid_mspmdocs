Completed tickets are not showing in Billing Batch

Last Modified

Fri Apr 12 14:32 GMT 2019

Description

  • Completed tickets are not showing in Billing Batch

Environment

Solution

  • A completed ticket will show in a billing batch only when it has been completed within current billing period or when date of Service Item is active
  • To see tickets that were completed in another period run a new billing batch:
    1. Run billing batch
    2. Click on Filter Service Items
    3. Enter a value for "Billing period contains" or enter a date range
  • NOTE: In order for a ticket to show in a billing batch it will need to contain an amount
