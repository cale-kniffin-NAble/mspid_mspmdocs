Can end-users attach files to tickets before they are created by a user?

Last Modified

Fri Jun 22 18:08 GMT 2018

Description

  • Can end-users attach files to tickets before they are created by a user?

Environment

Solution

  • End-user could potentially attach a file to an email that they would send to a specific email address that you use in for email retrieval
  • This would be the only way they could do that
  • Email would then be converted into a Ticket Request and end-user would not be able to attach any files further until the Ticket Request is converted into an actual support ticket by a user
