How to perform a user sync in

Last Modified

Description

  • How to perform a ser sync in

Environment

  • User Management

Solution

  • To perform this, will need to be integrated with your Dashboard
  • A user sync can be done by doing the following:
    1. Select Settings, found in the top right-hand corner
    2. Select Account Settings
    3. Under User Management, select Users
    4. Select Actions
    5. Select Synchronize MSP RMM
  • NOTE: You will know this has completed when it shows "Synchronisation completed"
