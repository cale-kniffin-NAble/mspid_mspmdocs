Meaning of creating a Project in

Last Modified

Mon Mar 18 12:48 GMT 2019

Description

  • This article provides information about the meaning of creating a Project in .

Environment

Solution

  • Creating a project within is a way to group similar tickets without affecting ticket progression or routing
  • Once that is associated with a ticket, you will be able to run a report on the Project Name to see how many tickets were associated with that project
  • To create a new project
    1. Click Customers, then the customer name
    2. Click Customers dropdown menu to see options available for the customer selected
    3. Click Projects, then Add Project
    4. Name the project, then click Add
  • To view a report of projects, create a Ticket report through the Reports menu, then filter by Customer and by project
