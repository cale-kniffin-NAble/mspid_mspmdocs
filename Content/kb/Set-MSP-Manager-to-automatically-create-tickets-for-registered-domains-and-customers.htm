Set to automatically create tickets for registered domains and customers

Last Modified

Wed Nov 30 21:47 GMT 2016

Description

This article describes how to set to automatically create tickets for registered domains and customers.

Environment

All versions

Solution

  1. Verify that the proper domain is selected for each customer, as it will be used to identify email matches to certain domain. For example, user@domain.com email can be associated to certain customer, if the customer is set to work with domain.com.
    1. Go to Customers and select the customer.
    2. Click Edit Customer.
    3. Populate the email domain.
    4. Click Save.
  2. Verify that you have one default service item for the customer site:
    1. Go to Customers and select the customer.
    2. Click Service Item.
    3. click Create/Edit Service Item.
    4. Select Default Service Item.
    5. Click Save.

The next time receives email from user@domain.com, the ticket will be created automatically under the specific customer.
