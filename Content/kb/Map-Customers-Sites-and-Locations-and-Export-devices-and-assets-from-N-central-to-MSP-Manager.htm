Map Customers Sites and Locations and Export devices and assets from to

Last Modified

Mon Mar 04 18:03 GMT 2019

Description

  • How to map Customers Sites and Locations
  • How to export assets/devices to .

Environment

  • 10.1 or later

Solution

  1. Set up a Default Asset Status in before importing devices from .
    1. Login to and go to Setting in the left menu
    2. Click on Assets > Asset Statuses > Add New Asset Status
    3. Enter Asset Status information, click Default Asset Status checkbox if appropriate, then click Save
**Note: Common Asset Status are "Active or Decommission" Create both and set Active as the Default Status
  1. Login to and go to the Service Organization level
  2. Enable PSA Integration with
    1. Administration > PSA Integration > Configure PSA Integration
    2. Select from the PSA Solution dropdown
    3. Enter credentials in the User Name and Password fieldsEnter Email Recipient, click Save
      • My credentials aren't accepted when trying to integrate and
  3. Map Customers from to
    1. Administration > PSA Integration > Customer Mapping
    2. Select a Customer from the dropdown
    3. Select a Location
    4. Click Save
  4. Create an Export profile
    1. Administration > PSA Integration > Export Devices
    2. Click Add
    3. Name the Export Profile
    4. Enter appropriate settings Settings and Targets tabs
    5. Click Save
  5. Export devices
    1. Administration > PSA Integration > Export Devices
    2. Click Export Now in the Action column
    3. Click Refresh Now
