When adding an email address as a Ticket Mailbox, the following error is given "Email Address is already in use"

Last Modified

Thu Jul 25 15:19 GMT 2019

Description

  • When adding an email address as a Ticket Mailbox, given the following Error : Email Address is already in use
  • In the har file, it shows a 409 POST error to https://api.mspmanager.com/api/monitoredmailbox

Environment

  • Ticket Mailbox

Solution

  • The reason for this is because the email address is already used for an account, most likely a customer contact or Integration part
  • Please edit this address for the contact to allow you to use this as a Ticket Mailbox
  • If you're having difficulties in locating this address, please contact Technical Support

Article Number: 123651

Internal notes: To locate this user, please do the following:1) Select Utility App2) Select Email Search3) Search for the email address supplied by the Case Contact4) Advise Case Contact which customer the email is associated with
