Error: 'error-unique-ownerid' when Quickbooks Desktop fails to integrate

Last Modified

Thu Oct 31 08:23 GMT 2019

Description

  • Quickbooks Desktop fails to integrate with ServiceDesk/
  • Integration attempt returns Error: 'Error QBWC1039'
  • Integration attempt returns Error: 'Error Unique OwnerID/FileID pair value required'

Environment

  • Quickbooks Desktop

Solution

  • The QWC file will need to be edited and the integration process attempted again:
  1. Open the QWC file in a text editor such as Notepad
  2. Find the OwnerID string
  3. Change one character in this string, letter or number
  4. Find the FileID string
  5. Change one character in this string, letter or number
  6. Save the file
  7. Attempt the integration process again
  • For addition information see this link : http://blog.accountinghelpline.com/quickbook/quickbooks-desktop-issues/qbwebconnector-error-qbwc1039-faqs-reasons-solutions/
