Create a contract that does not expire

Last Modified

Wed Nov 30 21:47 GMT 2016

Description

Can I create a contract that does not expire every year?

Environment

Solution

When you create a service item, select an end date that is a few years in the future to prolong the expiration.

Set the Service Item Notifications for expiration by clicking Settings > Notifications > Notification Settings > Service Item Notifications.
