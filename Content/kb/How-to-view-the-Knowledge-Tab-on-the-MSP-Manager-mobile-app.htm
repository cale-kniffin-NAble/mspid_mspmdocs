How to view the Knowledge Tab on the mobile app

Last Modified

Mon Feb 12 16:04 GMT 2018

Description

  • How to view the Knowledge Tab on the mobile app

Environment

  • Mobile App

Solution

  • The Knowledge Tab is under each customer, and needs to be accessed per customer
  1. Open the mobile app
  2. Select Customers > Customer name
  3. Select Knowledge tab
