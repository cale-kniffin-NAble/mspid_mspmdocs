How to set the From address for a customer in

Last Modified

Sun Oct 13 21:09 GMT 2019

Description

  • How to set the From address for a customer in

Environment

Solution

  1. Click on Customers > Click on the desired customer
  2. Click on Actions > Edit Customer
  3. Change the Send email from to a value in the dropdown
    • To add an email to the dropdown list, configure a Ticket Mailbox through Account Settings > Ticket Mailbox
  4. Select Save Customer
