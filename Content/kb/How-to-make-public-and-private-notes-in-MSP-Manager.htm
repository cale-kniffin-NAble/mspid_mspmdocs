Last Modified

Description

  • How to make public and private notes in ?

Environment

Solution

  1. Inside ticket, click on Add Notes tab
  2. Fill up description box for your notes
  3. Scroll down under attachment box
  4. Look for "Visible to the customer" option
  • If clicked on YES - note will be visible to customer
  • If clicked on NO - note will NOT be visible to customer
