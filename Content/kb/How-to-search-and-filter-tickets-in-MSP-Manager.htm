Description

  • How to search tickets in

Environment

Solution

  • Search tickets through Customer tab
    • Customer view is used to view tickets for only one Customer
      1. Click Customers tab
      2. Select the Customer associated with the Ticket you want to find
      3. Click the Tickets section
      4. Use filters to adjust ticket view
  • Search tickets through HelpDesk tab
    • HelpDesk is used to view tickets for all Customers
      1. Click HelpDesk tab
      2. Use filters to adjust ticket view
