How to add a new helpdesk ticket in

Last Modified

Tue Dec 05 14:20 GMT 2017

Description

  • How to add a new helpdesk ticket in

Environment

Solution

  1. Click Helpdesk
  2. Click Add a New Ticket
  3. Enter the ticket information
  4. Click Save Ticket, or Save and Start a Timer to start tracking your time.
  5. To add additional information to a ticket, click to the right of the ticket in the Helpdesk to access a list of ticket options.
