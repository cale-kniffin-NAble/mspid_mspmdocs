Unable to assign ticket to a User in

Last Modified

Wed Nov 21 06:54 GMT 2018

Description

  • Unable to assign ticket to a User in
  • User's name does not show up in the list

Environment

Solution

  • Ensure user is assigned these Customers
    1. Log into
    2. Go to Settings
    3. Go to User Management > Users
    4. Edit User
    5. Under Assigned Customers add customer(s) to manage
    6. Click Save Changes
