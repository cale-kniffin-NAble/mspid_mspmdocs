Last Modified

Description

  • How to add Users in

Environment

Solution

  1. In , go to Settings
  2. Click on Users & Permissions
  3. Click on Users
  4. Click Add User (top right)
  5. Fill out the necessary fields under USER INFORMATION / CUSTOMER ASSIGNMENT AND ACCESS
  6. Check the box saying I acknowledge I have read this.
  7. Click on the Save icon (Save Changes) on top
NOTE: For a newly created account, use the activation email link sent to the User's email to login for the first time.
