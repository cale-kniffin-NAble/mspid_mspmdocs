Notification Template not applying to tickets

Last Modified

Tue Mar 19 17:18 GMT 2019

Description

  • After editing the Notification template, changes didn't apply to the ticket

Environment

Solution

  • Ensure that the template was enabled
  1. Go to Settings > Templates > Notification Templates
  2. Select a Ticket Template
  3. Click Edit (pencil icon)
  4. Check box for Enable Custom Template
  5. Select the format type you want to be delivered "Plain Text" or "HTML"
  6. Click Save
