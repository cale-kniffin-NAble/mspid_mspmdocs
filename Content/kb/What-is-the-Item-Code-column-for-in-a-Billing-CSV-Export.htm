What is the Item Code column for in a Billing CSV Export?

Last Modified

Tue Jun 18 14:00 GMT 2019

Description

  • What is the Item Code column as seen when exporting the invoice as a CSV?

Environment

Solution

  • The Item Code Column is used when service items are integrated from services such as Quickbooks or Xero
  • When the column is populated this is the reference number from the integrated service item
