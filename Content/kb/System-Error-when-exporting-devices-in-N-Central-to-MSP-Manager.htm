System Error when exporting devices in to

Last Modified

Mon Oct 22 12:33 GMT 2018

Description

  • When trying to create an Export Devices in to , you get a system error

Environment

Solution

  • In order to export devices to , needs to be able to map them to a Status
  • This issue can occur when there are no Asset Statuses specified in
  1. Attempt to create a Profile for the device export to confirm that this is your issue
    • Note: You will see that the pick list for the Status of Devices after Export is blank
  2. Cancel out of the page
  3. To fix the issue, log in to
  4. Go to Assets > Add/Edit Asset > Asset Status
  5. Once you have added the new statuses, go back into and create a new Profile for the device export the Statuses will be there
