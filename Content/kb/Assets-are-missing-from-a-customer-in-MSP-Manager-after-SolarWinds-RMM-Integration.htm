Assets are missing from a customer in after Integration

Last Modified

Mon Dec 16 16:11 GMT 2019

Description

  • Assets are not being imported from after Integration into for some customers
  • Assets appear on , but not after integration

Environment

Solution

  1. Sign into https://app.mspmanager.com
  2. Select Assets
  3. Search for Asset using search bar or scrolling through Assets list
  4. Select targeted assets that aren't showing up under the right customer, select Overview
  5. Select Customer dropdown
  6. Enter customer under which you would like asset to appear
  7. If the assets doesn't exist then click on
  8. Add the asset then select the customer that own this asset
