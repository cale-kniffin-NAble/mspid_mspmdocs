Receiving "Something went wrong :: Alias not available" when opening portal website

Last Modified

Wed Oct 30 12:58 GMT 2019

Description

  • When logging into portal web page we receive "Something went wrong :: Alias not available"

Environment

Solution

  • Check and ensure correct portal url prefix is being used
    • To check portal url prefix go to Settings > Company Settings > Portal > Portal URL Prefix
      • If the wrong portal url is being used please attempt to use the configured portal url and attempt to log in again
      • Please Note: You will not be able to log into the portal with an user account. You will need to log in with a Contact account.
      • Once a contact has been added they will receive an email confirmation for Portal login if they were configured for Portal access
