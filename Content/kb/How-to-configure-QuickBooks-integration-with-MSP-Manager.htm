Last Modified

Description

  • Configuring QuickBooks Online / Desktop integration with

Environment

Solution

1.Disable the pop-up blocker in your browser
2.Start QuickBooks and click Lists > Item List
3. In , click > Account Settings > Integrations > QuickBooks Desktop / Online > Configure
4.Click Import QuickBooks Item "in case of QB Online"

Or Click Add a QuickBooks Item "in case of QB Desktop"

5.Refer back to the Item List in QuickBooks, and add an item in for each item; names must match exactly
6.Under Default Mappings, map each line item to the defaults listed, and click Save Default Mappings.

  • For more information, see: Accounting software integration.
