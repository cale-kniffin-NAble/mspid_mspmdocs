users not receiving notifications for tickets being created

Last Modified

Mon Oct 22 14:53 GMT 2018

Description

  • users not receiving notifications for tickets being created

Environment

Solution

  1. Log into
  2. Go to Settings > Company Settings > Notifications > Account Settings > Notifications > "When a new ticket gets created..."
  3. Click box for "Notify these specific users:"
  4. Choose the name of users that need to be notified
  5. Click Save Settings
