How to export billing invoices as a CSV file?

Last Modified

Mon May 13 14:26 GMT 2019

Description

  • How to export billing invoices as a CSV file?

Environment

Solution

  • If you are not using QuickBooks Desktop, QuickBooks Online, or Xero, you have the option to export your invoices as a CSV file.
  1. Navigate to the Billing tab
  2. Click New Billing Batch
  3. Click each service item you would like to add to the billing batch, and select Include
  4. Enter a New Batch Name
  5. Click Run Billing Batch
  6. Review the bills, and select each one you would like to invoice. Use the Select All option to quickly select every invoice in the list
  7. Select One Invoice per Customer if you would like to combine multiple bills into a single invoice for each customer
  8. Click Generate Invoices
  9. Select each invoice you would like to export, and choose CSV File as the export format
  10. Click Export Invoices to generate the CSV file
