Assigned Users to Ticket Queues functionality

Last Modified

Mon Jan 07 10:44 GMT 2019

Description

  • What effect does assigning users to a ticket queue have?

Environment

Solution

  • Ticket queues are essentially only a filtering option within and the assigned users have no impact
  • Assigned Users will not be notified or auto assigned tickets that are assigned to that queue
  • All users can view the ticket queue, it is not restricted to assigned users
