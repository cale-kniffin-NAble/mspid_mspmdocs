Insufficient access to add Asset Status in despite being administrator

Last Modified

Mon Oct 22 13:41 GMT 2018

Description

  • Insufficient access to add Asset Status in despite being administrator

Environment

Solution

  • By Default, an administrator does not have sufficient privileges to add Asset Status
  • This will need to be enabled following these steps:
  1. Login to
  2. Go to Settings > Company Settings > Roles
  3. Click Edit for Administrator role
  4. Click MSP Company Management to expand tab
  5. Click box to enable "Configure Asset Statuses"
  6. Click Save Role
