Is there a report to check the status of the hour retainer created in a service item?

Last Modified

Mon Jun 03 07:31 GMT 2019

Description

  • Is there a report to check the status of the retaining hours created in a service item?
  • A report that will show the remaining time in the hour retainer set by customer

Environment

Solution

  • There is no report that will show the hours spent in the Hour Retaining Service Item
  • As a workaround, you will be able to check the progress of the hour retaining service item
  1. Select a Particular Customer
  2. Select Service Item
  3. Expand the Hour Retaining Service Type by clicking the (+) button
