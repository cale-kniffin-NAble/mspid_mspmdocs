When trying to export Invoices to Xero, got prompted with Item code 'Example' not valid.

Last Modified

Thu Jul 09 23:15 GMT 2020

Description

  • Recently have modified the Rate and Service Item.
  • When checking the Mapped Items, it shows it's mapped correctly.
  • The item code exists in Xero and matched the Inventory Item.

Environment

  • Xero Accounting Software

Solution

This happens when the Inventory Item in M and the Item Code in Xero do not match.Ensure to check the following:
  • The entry code in Xero and are the same(Its cases sensitive).
  • Review the Tickets in the Billing Batch and ensure that the Tickets are using the correct Service Item.
  • Ensure as well that the Rate template is linked to the correct Mapped Items, else this yields to the error.
