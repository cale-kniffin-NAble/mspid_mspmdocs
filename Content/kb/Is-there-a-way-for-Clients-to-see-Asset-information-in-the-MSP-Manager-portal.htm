Is there a way for Clients to see Asset information in the portal?

Last Modified

Tue Sep 18 15:17 GMT 2018

Description

  • When a client logs into the Portal, it seems they can only submit support tickets
  • Is there a way for a client to see assets, servers, and networks, or is it strictly to see support tickets?

Environment

Solution

  • Portal Users and/or Portal Admins will not have the visibility to view Asset information, Networks or Devices
  • Only Users are able to view Assets in the dashboard
