Can I change the frequency that I receive daily notifications regarding expiring Service Items?

Last Modified

Wed Feb 20 18:21 GMT 2019

Description

  • Daily notifications are received regarding expiring service items
  • I would like these to be less frequent

Environment

Solution

  • Each day the service checks for expired or soon-to-expire items, if it matches the criteria set in the threshold, then a notification will be sent
  • You will continue to get notified each day until the criteria for expiry is no longer true
  • Currently there is no functionality to change the frequency of these notifications
