Description

  • How to set a rate as the default for a Customer in ?

Environment

Solution

  1. Log into
  2. Click on Customers tab
  3. Click on Customer to review
  4. Click on Rates tab to left
  5. Click edit option for Rate you want to make Default
  6. Under Status click box for Default Rate
  7. Click Save
