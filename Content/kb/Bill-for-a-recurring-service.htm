Bill for a recurring service

Last Modified

Tue Feb 25 10:55 GMT 2020

Description

  • Set up billing for a monthly/quarterly/yearly service
  • How to bill for a recurring service

Environment

Solution

  • Flat fee contracts will bill a specified amount for each asset that fits a criteria
    • Example: bill $5 per month for each device with antivirus enabled
  • Flat fee contracts do not need tickets attached in order to appear as billable in a billing batch
  • Multiple flat fee contracts can be created to bill for different services
  • To set up a Flat Fee contract for a customer
    1. Click Customers tab
    2. Select the customer's name
    3. Click Service Items
    4. Click Add a new Service Item
    5. Select Service Type > Flat Fee
    6. Name and configure the contract
      • "Use Asset Count" option will automatically count assets in that fit the specified criteria
      • Associate an Asset with a service item by editing the Asset and selecting the service item
      • More information on Assets within can be found here
    7. Click Save
    8. Run a billing batch which includes the flat fee service item
