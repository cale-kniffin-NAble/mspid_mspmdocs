Description

  • Duplicate assets are found in the Asset report in

Environment

Solution

  • To remove the duplicate assets, please do the following:
  1. Select Customers
  2. Select the Customer in question which has the duplicate asset
  3. Select Assets
  4. Select the appropriate column header to sort
  5. Select the duplicate asset in question you wish to remove
  6. Click the trash can icon
    in the right corner of the asset
