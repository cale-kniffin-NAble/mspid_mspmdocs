Cannot add Customer in

Last Modified

Mon May 27 14:28 GMT 2019

Description

  • When attempting to add a Customer in receive an error indicating that you cannot add client as they already exist

Environment

Solution

  • Reason for this is because we never permanently delete customers from database in case something was removed by mistake
  • Click on "Customers" in you'll notice a filter on this page that says "Showing active customers"
  • Change this to "Show all customers" which will show your deleted customer
  • You'll need to edit old customer name and potentially email domain you used so that you don't get duplicate errors and are able to create new Customer
  • Or you can just "Undelete" old Customer
