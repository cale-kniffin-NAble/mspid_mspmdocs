Export all ticket data

Last Modified

Fri Nov 03 15:38 GMT 2017

Description

Looking to export ticket data into Excel, into CSV format.

Environment

Solution

Ticket data can be exported to a CSV file format "Excel sheet" or PDF in the following manner:

1. Reports > Tickets Report
2. Choose the Customer and Service Items

  • Also can export time entries and Amounts in the Report > Apply Filter
