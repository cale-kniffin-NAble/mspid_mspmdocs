How to edit Mail Template for notification

Last Modified

Fri Jun 21 15:55 GMT 2019

Description

  • How to edit email notification for ticket statuses

Environment

Solution

  • Notifications can be edited by following the steps below:
  1. Click Settings
  2. Under Notifications > Notification Template
  3. Expand which template you wish to change
  4. Select language and edit Plain Text or HTML
  5. Make the desired changes
  6. Check the option for Enable Custom Template
  7. Click Save
