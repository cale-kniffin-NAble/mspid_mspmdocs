Duplicate email addresses are appearing in Optionally Notify Users dropdown when creating time entries

Last Modified

Wed May 01 09:58 GMT 2019

Description

  • Duplicate email addresses are appearing in Optionally Notify Users dropdown when creating time entries for tickets

Environment

    • Ticketing

Solution

  1. Select Customers, and select customer displaying duplicate email addresses
  2. Select Locations
  3. Expand locations by selecting plus sign
  4. Delete duplicate entries for contacts by selecting trash can icon to Delete Contact, and select YES, I WANT TO DELETE
