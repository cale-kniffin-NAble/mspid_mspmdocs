notification for expired service items

Last Modified

Wed Jul 17 15:56 GMT 2019

Description

  • How to set up notifications for expiring Service Items
  • How to change expiry date for Service Items

Environment

Solution

  • When a Service Item is going expire, then it should notify the user(s) based on the configuration set detailed below
  • Once the threshold is met, then an expiry notification is sent to the selected user(s)
  • To set this up, please do the following:
  1. Log in to
  2. Click on Settings
  3. Select Notifications > Service Item Notifications
  • To increase the expiry date for a Service Item, please do the following:
  1. Select Customers
  2. Select customer in question
  3. Select Service Items
  4. Search for the Service Item in question
  5. Click Edit Service Item
  6. Amend End date to new value
  7. Click Save Changes icon
