Unable to create a Customer in

Last Modified

Mon Oct 22 15:19 GMT 2018

Description

  • Administrator account is unable to add a Customer
  • Account has the proper roles to create Customers
  • Account is not part of any Access Group

Environment

Solution

  • Add the account to a Access Group:
  1. Sign into
  2. Go to Settings > User Management > Users
  3. Select desired user and click the Edit User icon
  4. Find Customer Access Group and add the default All Customers selection or any groups you created
  5. Check box "I acknowledge I have read this"
  6. Click Save Changes
