Last Modified

Description

  • How to run a report for all tickets with a specific time range in ?

Environment

Solution

  1. Log into
  2. Click on Reports tab
  3. Click on Tickets Reports
  4. Under Ticket Completed Date select Custom Completed Date
  5. Specify date range
  6. Click OK
  7. Click Apply Filters
