How to add a company logo to notifications in ?

Last Modified

Thu Apr 25 15:56 GMT 2019

Description

  • How to add a company logo to notifications in ?

Environment

Solution

  1. Go to Settings
  2. Open the Notifications tab
  3. Select Notification Templates
  4. Edit the desired template (Pencil icon):
  5. Use a Base64 image encoder to convert your company logo to an HTML string
  6. Copy the HTML code that is generated by the image encoder and paste it into the body of the notification template editor
  7. Click Preview to view the location of your logo in the notification
  8. Adjust the location of the HTML image code within the body of the template editor to change the placement of your logo
  9. Check Enable Custom Template
  10. Click Save
