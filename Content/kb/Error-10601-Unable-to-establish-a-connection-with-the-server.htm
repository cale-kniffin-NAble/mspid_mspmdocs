Error: 10601 Unable to establish a connection with the server

Last Modified

Mon Oct 22 13:26 GMT 2018

Description

  • This article provides brief information on the following error when creating a ticket from notifications:
    • 10601 Unable to establish a connection with the server

Environment

  • PSA Integration

Solution

  • This error can be safely ignored if tickets are able to be generated
  • However, if you cannot create tickets at all and you receive this error constantly under device's Audit Trail (top right corner under device level in ) make sure that can actually reach on this addresses on port 443:
    • mspmanager.com
    • api.mspmanager.com
  • NOTE: There is a third party tool called "Paping" that you can find in the web
  • You can use to ping the above sites (like a ping commend) but vs a custom port, such as 443 to see if you are getting packet loss
  • It works very similar to ping and this is the commend you can use to test:
    • paping.exe mspmanager.com -c 50 -p 443
