Quickbooks Desktop Web connector failing to login to

Last Modified

Fri Aug 16 20:40 GMT 2019

Description

  • Quickbooks Desktop Web connector failing to login to with message:
    • Error: "Invalid Password for username: <username>"

Environment

Solution

  1. Go to login screen
  2. Click on Click here to reset your password
  3. Enter e-mail address
  4. In notification received click on reset password and provide new password for user
  5. In Quickbooks Desktop Web Connector click File > Clear all stored passwords
  6. Change invalid password to one which was just updated
  7. Select checkbox next to application name and click on 'Update Selected'
