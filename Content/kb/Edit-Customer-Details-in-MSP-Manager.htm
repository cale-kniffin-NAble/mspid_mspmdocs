Last Modified

Description

  • This article provides steps to edit Customer Details in .

Environment

Solution

  1. Log in to
  2. Click Customers tab, and then select the Customer you want to edit.
  3. On the Customer dashboard, click the ACTIONS dropdown in the upper right and select Edit Customer
  4. An Edit Customer window will appear where you can edit the following:
    • Customer Name
    • Status
    • Assigned Users
    • Main Phone
    • Customer Since
    • Lead Source
    • Email Domain
    • Accounting Reference Name
    • Customer Reference ID
    • Customer to Invoice
    • Customer is Taxable option
    • Send Emails From
  5. Save your changes.
