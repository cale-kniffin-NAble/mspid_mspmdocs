Device Status option is empty during configuration

Last Modified

Mon Oct 22 12:44 GMT 2018

Description

  • On Device Status when configuring Export Devices Profile, the drop down is empty with no options to choose from

Environment

Solution

  • Asset Status needs to be added to :
  1. Go to Assets > Add Asset > Asset Status
  2. The entry will reflect in Central and the drop-down option should now be available
