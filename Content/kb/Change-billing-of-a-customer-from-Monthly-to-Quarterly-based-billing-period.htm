Change billing of a customer from Monthly to Quarterly based billing period

Last Modified

Wed Apr 26 13:13 GMT 2017

Description

The partner would ask "I have a client that I setup last month with a hour retainer service item. They would like us to bill them quarterly. I went it move the start date from 3/1/17 to 4/1/17 so that we can bill with the quarterly breakdown for our company. I'm getting an alert that says that we can change the start date. 1) Why is that? 2) What do I need to do fix this?"

Environment

Solution

Create a new service item that would start this month and run on a quarterly basis, then transfer existing tickets from a different service item to this new one as long as that old service item is active
