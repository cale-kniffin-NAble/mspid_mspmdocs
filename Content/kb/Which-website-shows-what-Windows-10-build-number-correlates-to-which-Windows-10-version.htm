Which website shows what Windows 10 build number correlates to which Windows 10 version

Last Modified

Wed Apr 10 10:46 GMT 2019

Description

  • Which website shows what Windows 10 build number correlates to which Windows 10 version
  • The installed Windows 10 build number is showing 10.0.15063, which Windows 10 version does this relate to (1703, 1709, 1803, 1809)

Environment

Solution

  • The last five digits of the Windows 10 build number relates to the specific version, i.e. 1703, 1709, 1803 etc
  • For example:
    • 10.0.15063 is the build number for 1703
    • 10.0.17736 is the build number for 1809
  • More information on this, and how to determine which build number is what Windows 10 version, please go here:
    • https://docs.microsoft.com/en-us/windows/windows-10/release-information
