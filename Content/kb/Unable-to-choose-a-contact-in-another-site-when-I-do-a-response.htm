Unable to choose a contact in another site when I do a response

Last Modified

Wed May 23 14:03 GMT 2018

Description

  • Unable to choose a contact in another site when I do a response

Environment

Solution

  • When adding a new response the TO section will only show contacts for which Location is selected when ticket is created
  • If no Location is selected then All Locations and All Contacts will be visible when adding a new response
  • To ensure All contacts and All locations are visible when adding a new response regardless if a Location was selected for ticket please vote/comment on this idea at http://ideas.n-able.com/a/dtd/Include-all-contacts-and-locations-when-adding-a-new-response/310913-4178?submitted=1#idea-tab-comments
