Auto assign a contact to CC field for all ticket responses for a Customer in

Last Modified

Mon Jan 20 10:38 GMT 2020

Description

  • Is it possible to setup our tickets to auto-populate the Contact CC entry?
  • We have a client POC who wants to be added to every ticket we create

Environment

Solution

  • This is currently not an option at this time in
  • The workaround for this would be to set the contact as a "Portal Admin"
  • A Portal Admin can then view all tickets submitted by their company on the portal
  • To configure the Portal, please do the following:
    1. Select Settings
    2. Select Company Settings
    3. Select Portal
    4. Update the URL in the Portal URL Prefix section
    5. Select Save Settings
