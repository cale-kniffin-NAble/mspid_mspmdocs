Cannot remove all assets from RMM ticket routing

Last Modified

Mon Oct 21 13:22 GMT 2019

Description

  • When trying to remove the assets from the RMM ticket routing it saves but the changes do not stay
  • How to prevent RMM tickets being created in but cannot remove the assets in the ticket routing

Environment

  • RMM PSA Integration

Solution

  • This is by design as removing the assets from the routing would cause no tickets to be generated
  • To prevent tickets being generated you will need to remove the PSA Integration from RMM
  • NOTE: Any other changes made whilst trying to clear the asset tab will also not be saved
