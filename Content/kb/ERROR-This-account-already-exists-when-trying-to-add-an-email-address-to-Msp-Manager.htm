ERROR: "This account already exists" when trying to add an email address to

Last Modified

Fri Sep 13 14:46 GMT 2019

Description

  • ERROR: "This account already exists" when trying to add an email address to

Environment

Solution

  • The email address cannot be added to the account because it already exists as the email address of an existing user, Customer Email Domain, Monitored Ticket Mailbox, etc.
  • Duplicate email addresses cannot exist in because it will cause issues with configuration.
  • To add the email address to the new user, Monitored Mailbox, etc., delete it from the original user, location, Customer, etc. and then add the email address to the new one.
