Report on time entry details

Last Modified

Wed Nov 30 21:47 GMT 2016

Description

Some of your customers would like a report on every time entry, and the details that go along with it. This is handy to show your customer the actual work that is being done for them, and you can also include the costs. So if one of your techs logs some onsite work and inputs a time entry of 2 hours, with details on what he did, we can report on all of these things from specific time periods.

Environment

All versions

Solution

  1. Login to and go to the "Reports" menu.
  2. Under "Ticket Reports," select the "Timesheet" report.
  3. You can then filter by a specific Technicians tickets, by Customer, completed data, etc. Set your filter and then select "Apply Filters" to generate the report:
