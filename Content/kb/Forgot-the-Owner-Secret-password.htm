Forgot the Owner Secret password

Last Modified

Wed Nov 30 21:47 GMT 2016

Description

This articles provides information when you have forgotten your own secret password.

Environment

All versions

Solution

Unfortunately, if you have forgotten owner secret password, you will not be able to recover it and will not be able to recover it for you as they are encrypted, by the same owner secret.

However, you may be able to delete the passwords that were based on owner secret if you do not need them anymore.

If you need to delete passwords, open a support case.
