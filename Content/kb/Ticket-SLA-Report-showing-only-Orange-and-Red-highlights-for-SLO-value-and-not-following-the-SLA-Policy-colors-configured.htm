Ticket SLA Report showing only Orange and Red highlights for SLO % value and not following the SLA Policy colors configured

Last Modified

Fri Jan 31 06:24 GMT 2020

Description

  • Report is not following the SLA Policy colors configured for the account.

Environment

Solution

  • The Available colors in the SLA report only contains two colors.
    1. Orange when SLA is < 100%
    2. Red when SLA is > 100%
