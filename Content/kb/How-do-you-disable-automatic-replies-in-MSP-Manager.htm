Last Modified

Description

  • When an automatic reply is sent to a customer, their ticketing system sends back an automatic reply
  • How do you disable automatic replies in

Environment

Solution

  • To disable the automatic replies, please do the following:
    1. In your control panel, select Settings
    2. Select Notifications
    3. Select When a Ticket Request is received...
    4. Enable Notify the person who submitted the ticket
    5. Select Save Settings
