error: A user with that email address already exists

Last Modified

Fri Oct 19 19:59 GMT 2018

Description

This article provides brief information and steps to resolve the following error when adding a user to their instance:

A user with that email address already exists

Environment

Solution

  • Possible causes are:
    • Partner has that email address associated to another instance (most likely a trial)
    • Partner used that email for the integration
  • Partner has that email address associated to another instance (most likely a trial):
  • For cases like this, we need to invalidate the email address on the other instance (like changing the domain or username)
    • Note: It's recommended not to delete that contact on that instance but just to invalidate the details
  • Partner used that email for the integration:
    • For cases like this, we need to change the email address being used for the integration
