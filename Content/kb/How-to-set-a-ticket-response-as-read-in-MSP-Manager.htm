How to set a ticket response as read in ?

Last Modified

Tue Mar 26 13:58 GMT 2019

Description

  • How can I set a ticket response as read in ?

Environment

Solution

  1. Open any desired ticket
  2. Select the response tab, then select the response
  3. In the top right of the response click the three dots
  4. Click Mark as Read
