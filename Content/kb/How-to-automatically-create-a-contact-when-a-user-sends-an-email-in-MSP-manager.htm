How to automatically create a contact when a user sends an email in

Last Modified

Fri May 17 15:25 GMT 2019

Description

  • How to automatically create a contact when a user sends an email in

Environment

  • SolaWinds

Solution

  • A contact can be automatically created if the domain of the sender's domain is added in the email domain
  • To add a domain for a specific customer
    1. Navigate to the specific customer
    2. Expand ACTIONS
    3. Select Edit Customer
    4. Add the domain the Email Domain field
