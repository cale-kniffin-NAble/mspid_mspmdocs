Start a Take Control remote session to N-Central integrated assets.

For accounts integrated with N-Central, you can remotely access your customers' devices from your Asset list as well as from Workspace pages. This can help you take control of these assets when you need to perform complex tasks, or access advanced features quickly. Start a remote session from three different locations in your account.

Enable remote access to assets

Before getting started, make sure the necessary assets are enabled for remote access.

  1. Select Assets from the left navigation pane and add a new asset or click to edit an existing asset in the list.
  2. Navigate to the RMM Features... section of the asset editor and click to expand it's options.
  3. Select Remote Control from the available options.
  4. Additionally, you can access the same options from the Asset List Widget in your Workspace by clicking the ellipsis next to an asset and selecting Edit.

  5. Save the asset to enable remote control.

Launch remote access from the Asset List

After enabling remote control on the necessary assets, you can start a remote session directly from the Asset List.

  1. Select Assets from the left navigation pane and locate an N-Central asset in the list.
  2. Click the Take Control icon in the Actions column to launch Take Control and start a remote session.

Launch remote access from a ticket

After adding an asset that is configured for remote access to a ticket, select the ticket from either a customers ticketing section or from the Ticket List in your Workspace.

  1. In the ticket containing the remote control enabled asset, click the Take Control link at the top of the ticket editor and select the appropriate asset.

Launch remote access from Workspace Assets

Make sure you have added the Asset List and Asset Details Widgets to your Workspace Page before getting started.

  1. Navigate to your Workspace and click Edit Workspace in the top right corner of the Workspace Page.
  2. In the Asset Details Widget, click the gear iconto configure it's options.
  3. Set the Linkage option to Yes, add the Asset List Widget to the Select widgets to link section and save the widget.

    4. With the two widgets linked, you can select an asset from the Asset List and its information will populate in the Asset Details widget.

  1. Click the Take Control icon in the top right corner of the Asset Details Widget to launch a remote session directly from the Workspace page.
  2. The same functionality also exists in the Asset list. Edit the widget to adjust it's size to show additional columns, select an asset from the list and click the Take Control icon next to the asset to launch Take Control.
