Single Sign-on

The Single Sign-On service allows users to access our supported product portfolio through a single login. After the user's credentials are registered with the Single Sign-On service, a single login is used to access all supported SolarWinds products.

Single Sign-On has the following benefits:

  • Provides the user with full control over their login.
  • Makes logging into multiple products much easier - the user only has to remember one username and password.
  • Improves security - login credentials are only entered once in the Single Sign-On service, which leads to reduced exposure of usernames and passwords.
  • Makes it easier to apply the companyâ€™s login policy as there is only one login to manage.
  • Simplifies switching between products through the Navigator bar.
  • Increases productivity as users spend less time attempting to login as well as potentially reduce the administrative burden of dealing with lost credential requests.
  • Encourages the use of more complex passwords as there is only one login to remember.
