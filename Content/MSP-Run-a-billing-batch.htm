Run a billing batch

  Watch a video on how to bill a customer.

  1. Click Billing.
  2. Click New Billing Batch.
  3. Select which customers and services to include in the new billing batch:
    • Click Select All.
    • Click Filter Service Items to narrow down the customers and services.
    • Expand a customer in the list, and select Include.
  4. Name the batch, and click Run Billing Batch.
  5. Click Select All, or select Include to choose individual service items and create separate invoices.
  6. Select One Invoice Per Customer to combine each customer's billing into one invoice.
  7. Click Generate Invoice.
