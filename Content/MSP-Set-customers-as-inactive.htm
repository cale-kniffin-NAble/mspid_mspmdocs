Set customers as inactive

Question

I have a customer who no longer uses my services. How do I hide them from my active customer list, but not delete their information entirely?

Answer

  1. Navigate to and select a customer from the list.
  2. Select Actions > Edit Customer.
  3. Uncheck the Active checkbox.
  4. Select Save Customer.

If this customer hires you again, set their status back to active. First, locate them in the customer list:

  1. Navigate to and select next to the search field.
  2. Select Inactive Customers and select the customer you would like to re-activate.
  3. Select Actions > Edit Customer.
  4. Select Active > Save Customer.
