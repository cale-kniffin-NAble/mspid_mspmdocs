Start and pause timers from the ticket list

Quickly start a timer from the list of tickets without opening the full ticket.

  1. Click Helpdesk.
  2. Use the filters on the left to locate a ticket.
  3. Click the Play button to start a timer.
  4. Click the Pause button to put the timer on hold, and use the play button when you are ready to continue tracking time.
  5. Click the Stop button to stop the timer and enter a description of work.
  6. Save the time entry.

The time entry is added to the ticket along with the description of work. Continue using the timer to track your time and create additional time entries as needed.
